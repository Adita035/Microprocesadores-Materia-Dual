# Sistema de Gestión para Club Deportivo

## Descripción

Sistema web desarrollado para administrar de forma integral un club deportivo, permitiendo la gestión de usuarios, entrenadores, socios, actividades deportivas, instalaciones, reservas y material deportivo.

## Objetivo

Desarrollar una plataforma que facilite la administración y el control de los procesos operativos del club deportivo mediante herramientas digitales accesibles y eficientes.

## Tecnologías Utilizadas

### Backend
- Kotlin
- Spring Boot
- Spring WebFlux
- Spring Security
- R2DBC
- MySQL

## Funcionalidades

- Gestión de usuarios
- Gestión de socios
- Gestión de entrenadores
- Gestión de actividades deportivas
- Reservas de espacios deportivos
- Control de inventario deportivo
- Generación de reportes
